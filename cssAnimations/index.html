Designing, Defining, and Triggering CSS3 Animations without Custom Libraries

Check out all of the animations you can do with CSS!

All animations can be set to start automatically or they can be set with a delay. It's all up to you!


You can even do just simple animations like the ones below.


Just like with transforms and transitions, each animation also has a timing delay and you can even specify how many times you'd like the animation to cycle through. Plus, you can choose how each animated object looks before and after the animation, whether you'd like to keep the new properties or revert back to the old ones.
